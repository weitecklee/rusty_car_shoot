# Rusty Car Shoot

This repository contains code generated for the [Car Shoot](https://github.com/CleanCut/rusty_engine/blob/main/scenarios/car_shoot.md) game scenario as part of the [Rusty Engine Tutorial](https://cleancut.github.io/rusty_engine/).

Additional assets are from [Ravenmore](https://ravenmore.itch.io/free-space-shooter-assets).
